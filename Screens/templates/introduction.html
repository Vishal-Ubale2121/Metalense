Objective of this toolkit

This toolkit aims to provide banks and financial institutions with an understanding as to where they currently stand in terms of preparedness to complete the KYC & onboarding of metaverse customers.

How should this toolkit be used?

Step 1: User should start first with the "Client Information" and enter the required information in the appropriate fields, and also specify the portfolio they wish to evaluate.
Step 2: User should then shift to "Initial screening" sheet and answer all the questions there to determine whether they should proceed with this toolkit.
Step 3: If the "Initial screening" sheet concludes that user should proceed with this toolkit, the user should shift to "Assessment Parameters" tab.
Step 4: There are 4 themes in this toolkit, each representing one focus area on which the stakeholder has to evaluate themselves. All questions are mandatory.
Step 5: The "Scoring guide" column in each sheet provides user with guidance on how to come up with the correct response based on their self-evaluation.
Step 6: Based on the scores assigned for each question, an overall score for the specific theme is indicated at the bottom of the sheet.
Step 7: Repeat the same set of actions for the remaining 3 themes.
Step 8: After answering all the questions and assigning the scores, user has to click "Results" to move to the dashboard.
Step 9: The dashboard will be automatically generated, based on which user can check their preparedness levels.
